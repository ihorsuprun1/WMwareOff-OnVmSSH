1. Для работы приложения на хосте WmvareExsi должно быть включено доступ по ssh
2. Приложение должно находится в одной сети с хостом WmvareExsi или еже иметь доступ по ssh к нему
3. Также вам понадобится саздать бота в телеграме и ввести его токен в файл конфигурации Token.json
4. Настройка о хосте WmvareExsi нужно будет внести в файл конфигурации Host.json
5. Настройка о пользователях какие имеют доступ к виключения виртуальніх машин  нужно будет внести в файл конфигурации UsersAcces.json

Приложение предназначено для включением и отключением виртуальніх машин через телеграмм
